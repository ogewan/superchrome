# SuperChrome

A powerful Chrome extension for tab and browser management.

## Status

Active development — tab management, URL manager, and grouping features built.

## Setup

1. Open `chrome://extensions/` in Chrome
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** and select this directory

No build step required — plain JS loaded directly by Chrome.
